# Yo!LOG

You Log Only Once

## Introduction

It is a simple (no frills) in-memory log viewer in browser.

Essentially, aim is to build `tail -f` equavalent for OpenTelemetry logs.

**Backend** will be implemented in Python.
**UI** will be implemented in Angular.
